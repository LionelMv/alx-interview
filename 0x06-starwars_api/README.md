# Star Wars API
## About
Includes scripts to print all characters of a Star Wars movie using the Star Wars API provided.

## Environment
- Ubuntu 20.04 LTS
- Node 10
- Style Guidelines:
    - [JavaScript Semi-Standard Style](https://github.com/standard/semistandard)
    - [Google Style Python Docstrings](https://sphinxcontrib-napoleon.readthedocs.io/en/latest/example_google.html)
    - [Pycodestyle](https://pycodestyle.pycqa.org/en/latest/intro.html#example-usage-and-output)
    - [Type Annotations](https://mypy.readthedocs.io/en/latest/cheat_sheet_py3.html)

## Resources
- [Request module](https://github.com/request/request)

### Install Node 10
```
$ curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
$ sudo apt-get install -y nodejs
```

### Install semi-standard
```
$ sudo npm install semistandard --global
```

### Install request module
```
$ sudo npm install request --global
$ export NODE_PATH=/usr/lib/node_modules
```
